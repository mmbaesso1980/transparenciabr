# Changelog de Melhorias Autônomas

## v1.1.0-alpha.1 (Ciclo Autônomo) - 2026-06-07

### ✨ Novas Features

- **Ferramenta `http_request`:** Adicionada a proposta de uma nova ferramenta para realizar chamadas HTTP genéricas (GET, POST, PUT, etc.), permitindo a interação com qualquer API RESTful. O código-fonte foi implementado em `aurora_v3_maestro/tools/http_client.py`.

### 📄 Documentação

- **Roadmap Autônomo:** Criado o documento `aurora_v3_maestro/docs/ROADMAP_AUTONOMO_V1.md` para guiar e registrar os ciclos de melhoria autônoma.
- **Proposta de Prompt:** Criado o arquivo `aurora_v3_maestro/prompts/PROMPT_EVOLUTION_PROPOSAL.md` com a definição exata da nova ferramenta para ser integrada ao System Prompt.
- **Changelog:** Este arquivo foi criado para manter um registro histórico das evoluções autônomas.
