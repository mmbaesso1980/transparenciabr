## Pontos de decisГЈo (precisam alinhamento Comandante)

1. **HQ separada ou plugin do AURORA HQ?**
   - Separada (prГіprio subdomain): clara separaГ§ГЈo Maestro vs AURORA
   - Plugin: reusa frontend existente, mais economia
   - **RecomendaГ§ГЈo:** separada. Maestro Г© arquitetura distinta (projeto-codex-br).

2. **Real-time vs polling?**
   - Firestore listeners: $$$ por documento lido por janela ativa
   - Polling 5s: mais barato, latГӘncia tolerГЎvel
   - **RecomendaГ§ГЈo:** listeners apenas para 3 docs (kill_switch, Гәltimo audit_event, Гәltimo snapshot). Polling pro resto.

3. **Avatares = quantos?**
   - 1 Maestro + 16-20 detetives (escala AURORA): ~6MB de sprites
   - 1 Maestro + 6 detetives (escala Maestro v1.0): ~2MB
   - **RecomendaГ§ГЈo:** 1 + 6 inicial. Adicionar mais conforme Maestro v1.1+ expande tools.

4. **Hosting Firebase ou Cloud Run?**
   - Firebase Hosting: SSR-less, custo zero idle, CDN global
   - Cloud Run: SSR possГӯvel, custo idle ~R$ 5/mГӘs
   - **RecomendaГ§ГЈo:** Firebase Hosting (SPA puro).
