## 1. BASE NORMATIVA — O QUE A CEAP PERMITE

### 1.1 Ato da Mesa nº 43 de 21/05/2009 (consolidado)

Fonte oficial: [Câmara — Ato da Mesa 43/2009](https://www2.camara.leg.br/legin/int/atomes/2009/atodamesa-43-21-maio-2009-588364-normaatualizada-cd-mesa.html). Reajuste 13,7% em 2026 ([Gazeta do Povo](https://www.gazetadopovo.com.br/economia/camara-dos-deputados-aumenta-verba-de-gabinete/)).

> **Art. 1º** — A CEAP destina-se a custear gastos **exclusivamente vinculados ao exercício da atividade parlamentar**, observados os limites mensais por estado (R$ 41 mil a R$ 57 mil em 2026).

> **Art. 4º** — O parlamentar **assume inteira responsabilidade** pela liquidação da despesa e atesta que ela é vinculada ao mandato.

> **Art. 4º §13** — É **vedado** ressarcir bens/serviços fornecidos por empresa em que o deputado ou parente até 3º grau seja proprietário ou detenha qualquer participação.

> **Art. 15** — **Não serão permitidos gastos de caráter eleitoral**.

> **§2º** — Excedentes ao saldo são **automaticamente deduzidos da remuneração** do parlamentar.

### 1.2 Tipos de despesa permitidos (taxonomia oficial)

A Câmara classifica os reembolsos em **13 categorias**:

| # | Tipo | Limite/Restrição |
|---|------|------------------|
| 1 | Manutenção de Escritório de Apoio à Atividade Parlamentar | locação imóvel, energia, água, internet, condomínio — só na UF do mandato |
| 2 | Locação ou fretamento de aeronaves | só transporte do mandato; fretamento exige justificativa |
| 3 | Locação ou fretamento de veículos automotores | exige nota fiscal com placa, KM rodado |
| 4 | Locação ou fretamento de embarcações | só estados com geografia que justifique |
| 5 | Combustíveis e lubrificantes | só veículos a serviço do mandato |
| 6 | Serviço de táxi, pedágio e estacionamento | sem caráter pessoal/lazer |
| 7 | Passagens aéreas (SIGEPA) | só trechos de interesse do mandato; auto-emissão |
| 8 | Telefonia | linhas vinculadas ao gabinete |
| 9 | Serviços postais | exceto cartões pessoais/eleitorais |
| 10 | Divulgação da atividade parlamentar | **vedado conteúdo eleitoral, propaganda partidária, promoção pessoal fora de relatório** |
| 11 | Consultorias, pesquisas e trabalhos técnicos | exige produto entregue, escopo delimitado |
| 12 | Alimentação do parlamentar | **só do parlamentar, não terceiros**; valor razoável |
| 13 | Hospedagem (exceto no DF) | em deslocamento de mandato |
| 14 | Cursos, palestras, eventos | vinculação com mandato |
| 15 | Assinaturas de publicações | jornais, revistas técnicas |
| 16 | Aquisição de material de escritório, suprimentos de informática | bens de consumo, não bens patrimoniais |

function crossCheckOcr(docAi, geminiVision) {
  const violations = [];
  if (Math.abs(docAi.valor_total - geminiVision.valor_total) / docAi.valor_total > 0.01)
    violations.push({ field: 'valor_total', severity: 'CRITICAL' });
  if (docAi.cnpj_emissor !== geminiVision.cnpj_emissor)

  if (docAi.data_emissao !== geminiVision.data_emissao)
    violations.push({ field: 'data_emissao', severity: 'HIGH' });
  return { ok: violations.length === 0, violations };
}

### 6.3 Query Protocolo F.L.A.V.I.O. completa

```sql
-- Detector de rachadinha estendido — F11 série completa
WITH SecretarioPerfil AS (
  SELECT
    s.id_lotacao,
    s.parlamentar_id,
    s.cpf_secretario,
    s.cargo,
    s.salario_bruto,
    s.endereco_residencial,
    s.reduto_eleitoral_chefe,
    s.uf_lotacao,
    -- F11.001
    IF(s.endereco_residencial != 'DF' AND s.uf_lotacao = 'DF', TRUE, FALSE) AS flag_distancia,
    -- F11.002
    IF(DATE_DIFF(s.data_exoneracao, s.data_nomeacao, DAY) < 60 AND s.data_exoneracao IS NOT NULL, TRUE, FALSE) AS flag_porta_giratoria,
    -- F11.004
    IF(s.flag_familiar_3grau, TRUE, FALSE) AS flag_nepotismo,
    -- F11.010
    IF(s.salario_bruto > (SELECT PERCENTILE_CONT(salario_bruto, 0.95) OVER ()
                          FROM `fiscallizapa.analytics.fato_secretario_gabinete`
                          WHERE cargo = s.cargo), TRUE, FALSE) AS flag_sobrepreco_salarial
  FROM `fiscallizapa.analytics.fato_secretario_gabinete` s
),
ChefeViajaParaSecretario AS (
  -- F11.028 — chefe (deputado) viaja >1x/mês ao reduto eleitoral do secretário
  SELECT
    s.parlamentar_id,
    s.cpf_secretario,
    COUNT(DISTINCT c.id_despesa) AS num_despesas_no_reduto_secretario,
    COUNT(DISTINCT DATE_TRUNC(c.data_emissao, MONTH)) AS meses_com_visita
  FROM SecretarioPerfil s
  JOIN `fiscallizapa.analytics.fato_ceap_despesa` c
    ON c.parlamentar_id = s.parlamentar_id
    AND c.tipo_despesa IN ('Combustíveis e lubrificantes', 'Locação de veículo', 'Hospedagem')
    AND ST_DWITHIN(
      ST_GEOGPOINT(c.fornecedor_lng, c.fornecedor_lat),
      ST_GEOGPOINT(s.reduto_lng, s.reduto_lat),
      30000 -- 30 km de raio
    )
  GROUP BY 1, 2
  HAVING meses_com_visita >= 6 -- >1x/mês recorrente
)
SELECT
  sp.parlamentar_id,
  sp.cpf_secretario,
  sp.cargo,
  sp.salario_bruto,
  -- Score F.L.A.V.I.O. composto
  ROUND(
    (CAST(sp.flag_distancia AS INT64) * 25) +
    (CAST(sp.flag_porta_giratoria AS INT64) * 20) +
    (CAST(sp.flag_nepotismo AS INT64) * 30) +
    (CAST(sp.flag_sobrepreco_salarial AS INT64) * 15) +
    (CAST(IF(cv.num_despesas_no_reduto_secretario IS NOT NULL, TRUE, FALSE) AS INT64) * 30)
  , 2) AS score_flavio,
  CASE
    WHEN score_flavio >= 75 THEN 3   -- LEVEL 3: rachadinha provável
    WHEN score_flavio >= 50 THEN 2
    WHEN score_flavio >= 25 THEN 1
    ELSE 0
  END AS flavio_level
FROM SecretarioPerfil sp
LEFT JOIN ChefeViajaParaSecretario cv USING (parlamentar_id, cpf_secretario)
ORDER BY score_flavio DESC;
```
